# Excel Project – Customer Demographics & Bike Purchase Analysis

## Project Overview
This project analyzes a customer demographic dataset to uncover insights related to bike-purchasing behavior.  
Using Excel tools such as PivotTables, PivotCharts, functions, and dashboard design, the project converts raw data into clear, actionable business insights.

---

## Dataset Information

**File:** `Excel Project Dataset.xlsx`

The dataset includes the following columns:

- ID  
- Marital Status  
- Gender  
- Income  
- Children  
- Education  
- Occupation  
- Home Owner  
- Cars  
- Commute Distance  
- Region  
- Age  
- Purchased Bike  

Each row represents a single customer with demographic and behavioral information.

---

## Objectives

- Clean and organize the dataset  
- Perform exploratory data analysis  
- Build PivotTables to identify patterns  
- Create an interactive dashboard  
- Understand factors influencing bike purchases  

---

## Excel Techniques Used

- PivotTables  
- PivotCharts  
- Data cleaning (formatting, corrections, standardization)  
- Logical functions (IF, COUNTIF, SUMIF)  
- Lookup functions (XLOOKUP, VLOOKUP)  
- Data validation  
- Slicers and filters  
- Dashboard creation  

---

## Key Insights (Example – Replace with Your Findings)

- Higher-income customers showed higher bike-purchase rates.  
- Ages 35–50 had the strongest likelihood to purchase a bike.  
- Commute distances between 2–5 miles correlated with more purchases.  
- Homeowners purchased more bikes than renters.  
- Professionals and skilled manual workers had higher purchase patterns.
